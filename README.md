# Travel-agency-website
This is a fully responsive and visually appealing travel agency website designed to provide a seamless booking experience for users. Built using HTML, CSS, and JavaScript, the website showcases different travel destinations, services, and an easy-to-navigate user interface. The project focuses on providing an engaging experience through smooth animations, hover effects, and a clean design.

Features:
This travel website includes multiple sections such as Home, Destinations, About Us, Services, and Contact, ensuring an organized flow of content. It is designed to be responsive, making it look great on mobile, tablet, and desktop screens. The website also includes smooth scrolling, interactive buttons, and attractive layouts to enhance user engagement.

Technologies Used:
The project is developed using the following technologies:
HTML – For structuring the content and defining page layouts.
CSS – To enhance the visual appeal with styling, animations, and responsiveness using Flexbox and Media Queries.
JavaScript – For adding interactivity like smooth scrolling and dynamic UI elements.
